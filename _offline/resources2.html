I have used multiple resources to accomplish the Node.js and MongoDB assignment. Some of these are from the Node for Beginners list and some I have found myself.

Resources from Node for Beginners

  • cwbuecheler.com tutorials - These are great because the writer addresses me like I have common sense, but need a little break down on what exactly all the technical stuff means and does. Couldn't have figured most of this out without these tutorials.
    • THE DEAD-SIMPLE STEP-BY-STEP GUIDE FOR FRONT-END DEVELOPERS TO GETTING UP AND RUNNING WITH NODE.JS, EXPRESS, JADE, AND MONGODB
    • CREATING A SIMPLE RESTFUL WEB APP WITH NODE.JS, EXPRESS, AND MONGODB
  • TutorialIndustry.com - Short and sweet resources to extend your learning.
    • Node.js Mongodb: Express, Mongoose Installation, Examples
    • 5 Best Node.js Books for Beginners- Javascript Basic, Nodejs Starter Books
  • flippinawesome.org - Really great information to understand the workings of Express.
    • Intro to Express.js Parameters, Error Handling and Other Middleware
    • Express.js Fundamentals

Additional Resources I have used

  • Marcus Node Bits: monk is an easy way to access MongoDb - In researching various ways of connecting MongoDB and Node, I found I just liked the way Monk worked better. I'm certainly still experimenting with Mongoose, but what I could make work with Monk would not when I rewrote it using Mongoose. I like the idea of the schema design of Mongoose, but haven't been successful in using it... yet!
  • Monk GitHub - If you want to learn about using Monk, here's the official GitHub that outlines exactly how to write queries and so forth using it. Very useful stuff!
  • MongoLab GitHub - Using MongoLab was so easy. These resources on their GitHub (in addition to the site documentation) really helped a lot. Especially check out this repo to get great examples of hooking a MongoLab DB either directly with Node or with Mongoose (or with many other languages outside of Node). I'm still using this resource to eventually move my app to using Mongoose instead of Monk.
